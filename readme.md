# Heroku Deployment Demo

## Requirements
1. Node.js v14+
2. NPM v6+
3. Heroku CLI v7+

### Heroku CLI

Use one of the installation methods documented [here](https://devcenter.heroku.com/articles/heroku-cli)

### Steps
1. `heroku login`
2. `heroku git:clone -a pt-3-heroku-demo`
3. `git push heroku main`
